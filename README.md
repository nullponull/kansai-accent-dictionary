# 関西弁アクセント辞書 (Kansai Dialect Accent Dictionary)

日本語の京阪式アクセント（大阪・京都方言）を収録した辞書データです。音声合成、言語学研究、方言学習などに活用できます。

## 📊 収録データ

- **収録語数**: 4,615語
- **アクセント体系**: 京阪式（Keihan-type）
- **形式**: CSV、MeCab辞書形式

## 🎯 特徴

### 京阪式アクセントとは

京阪式アクセントは、大阪・京都を中心とした関西地方で使用されるアクセント体系です。東京式アクセントとは異なる以下の特徴があります：

- **起式の区別**: 高起式（H）と低起式（L）の区別がある
- **アクセント核**: 下がり目の位置を示す
- **特殊拍**: 長音や撥音もアクセント核を担える

### アクセント記法

本辞書では以下の記法を使用しています：

```
形式: [H|L][数字列]
```

- **H/L**: 起式（H=高起式、L=低起式）
- **数字**: 各拍の高低とアクセント核の位置

**例**:
- `H00`: 高起式、核なし（例: はな「鼻」）
- `L02`: 低起式、2拍目に核（例: あめ「雨」）
- `H01`: 高起式、1拍目に核（例: はし「橋」）

## 📖 データソース

本辞書は以下のオープンデータを統合して作成されました：

### 1. 近畿日本語アクセント辞典（3,886語）

- **提供元**: 尾水坂本氏
- **URL**: https://sites.google.com/view/omizansakamoto/documents/teishouzan
- **収録語数**: 3,886語（全体の84.2%）
- **収集方法**: Google Sheets API経由でCSVダウンロード
- **ライセンス**: 公開データとして利用

### 2. Wikibooks 京阪式アクセント（基礎）（514語）

- **提供元**: Wikibooks
- **URL**: https://ja.wikibooks.org/wiki/京阪式アクセント（基礎）
- **収録語数**: 514語（全体の11.1%）
- **収集方法**: HTMLパース
- **ライセンス**: CC BY-SA 3.0

### 3. 甲種日本語アクセント辞典（215語）

- **提供元**: せきゆう氏
- **URL**: https://sekiyu-oh.com/accent/
- **収録語数**: 215語（全体の4.7%）
- **収集方法**: スクリーンショット + OCR（OpenAI Vision API）
- **ライセンス**: 公開データとして利用

## 🚀 使用方法

### CSV形式の辞書

```python
import csv

# 辞書を読み込む
with open('data/kansai_accent_dict.csv', 'r', encoding='utf-8') as f:
    reader = csv.DictReader(f)
    for row in reader:
        word = row['word']
        accent = row['accent']
        pos = row['pos']
        print(f"{word}: {accent} ({pos})")
```

### MeCab形式の辞書

MeCab形式の辞書の使用方法については、`mecab/README.md`を参照してください。
