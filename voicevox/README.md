# VOICEVOX/VOICEPEAK用 関西弁アクセント辞書

このディレクトリには、VOICEVOX/VOICEPEAK用に変換された関西弁アクセント辞書が含まれています。

## ファイル

- `kansai_accent_voicevox.csv`: VOICEVOX/VOICEPEAK形式のアクセント辞書（4,346エントリ）

## CSV形式

ヘッダー行なしのCSVファイルです。各行の形式：

```
表記,アクセント位置,読み,優先度
```

### 各列の説明

1. **表記**: 単語の表記（漢字またはひらがな）
2. **アクセント位置**: アクセント核の位置（0～文字数）
   - `0`: 平板型（下がり目なし）
   - `1`: 頭高型（1モーラ目の後で下がる）
   - `2以上`: 中高型（指定したモーラ目の後で下がる）
3. **読み**: カタカナ表記の読み
4. **優先度**: 0～999（数値が小さいほど優先度が高い、すべて0に設定）

### 例

```csv
音声,2,オンセイ,0
合成,1,ゴウセイ,0
愛,1,アイ,0
青,0,アオ,0
```

## 使い方

### VOICEVOX

1. VOICEVOXの設定から「アクセント辞書」を開く
2. 「辞書の追加」をクリック
3. `kansai_accent_voicevox.csv`を選択
4. 関西弁のアクセントが適用されます

### VOICEPEAK

1. VOICEPEAKの設定から「アクセント辞書」を開く
2. 「辞書をインポート」をクリック
3. `kansai_accent_voicevox.csv`を選択
4. 関西弁のアクセントが適用されます

## 注意事項

- この辞書は京阪式アクセント（関西弁）を基にしています
- 元の京阪式アクセント（H/L記号）から自動変換しているため、一部のアクセントは近似値となっています
- 複数のアクセントパターンがある単語は、最初のパターンのみを採用しています
- 接辞（-で始まる単語）は除外されています

## 変換について

この辞書は、`../data/kansai_accent_dict.csv`から`../tools/convert_to_voicevox.py`を使用して自動変換されました。

### 変換ルール

京阪式アクセント（H/L記号）からVOICEVOX形式（数値）への変換：

- **HH, HHH...**: 高起式で下がり目なし → `1`（頭高型）
- **LL, LLL...**: 低起式で下がり目なし → `0`（平板型）
- **H01**: 高起式で1モーラ目の後に下がる → `1`
- **L02**: 低起式で2モーラ目の後に下がる → `2`
- **HHL**: 高起式から低起式へ → `1`（頭高型）
- **LLH**: 低起式から高起式へ → `2`（2モーラ目の後に変化）
