## 1. 近畿日本語アクセント辞典

### 基本情報

- **提供元**: 尾水坂本氏
- **URL**: https://sites.google.com/view/omizansakamoto/documents/teishouzan
- **収録語数**: 4,869行（重複削除後3,886語）
- **形式**: Google Sheets（埋め込み）

### データ構造

| 列 | 内容 | 例 |
|----|------|-----|
| A | 単語 | ああ |
| B | 元表記 | - |
| C | 品詞 | 副 |
| D | アクセント | HH |
| E | 備考 | 言い切って体言的に扱うときはHL |
| F | 用法解説 | - |

### 収集方法

1. Google SitesのページからiframeのURLを抽出
2. Google Sheets ID: `1GVdsvJEQ9Mnl0AjBBOlHcWSUXaMiKxlSs9F6nzwr3gg`
3. Google Visualization API (`/gviz/tq?tqx=out:csv`)でCSVダウンロード
4. Pythonで重複削除とデータクリーニング

### ライセンス

公開データとして利用。商用利用の際は提供元に確認することを推奨します。

## 2. Wikibooks 京阪式アクセント（基礎）

### 基本情報

- **提供元**: Wikibooks コミュニティ
- **URL**: https://ja.wikibooks.org/wiki/京阪式アクセント（基礎）
- **収録語数**: 1,181語（重複削除後514語）
- **形式**: Wikibooks HTML

### データ構造

Wikibooksのページには、以下のセクションが含まれています：

- 1拍名詞
- 2拍名詞
- 3拍名詞
- 4拍名詞
- 動詞
- 形容詞

各セクションに表形式でアクセント情報が記載されています。

### 収集方法

1. ブラウザでページを開き、Markdownとして保存
2. Pythonで表データを抽出
3. 重複削除とデータクリーニング

### ライセンス

**CC BY-SA 3.0** (Creative Commons Attribution-ShareAlike 3.0 Unported)

- 表示: 適切なクレジットを表示する必要があります
- 継承: 同じライセンスで再配布する必要があります

詳細: https://creativecommons.org/licenses/by-sa/3.0/

## 3. 甲種日本語アクセント辞典

### 基本情報

- **提供元**: せきゆう氏
- **URL**: https://sekiyu-oh.com/accent/
- **収録語数**: 約4,894語（本辞書では215語を収録）
- **形式**: HTML表

### データ構造

| 列 | 内容 | 例 |
|----|------|-----|
| 単語 | 単語 | ああ |
| 元表記 | 漢字等 | - |
| 品詞 | 品詞 | 副 |
| アクセント | アクセント | HH |
| 備考 | 備考 | - |

### 収集方法

1. あ行〜わ行の各ページをブラウザでスクロール
2. 47枚のスクリーンショットを撮影
3. OpenAI Vision API (gpt-4.1-mini)でOCR処理
4. 表データを抽出してCSV形式に変換
5. 重複削除（近畿日本語アクセント辞典と重複する部分を除外）

### ライセンス

公開データとして利用。商用利用の際は提供元に確認することを推奨します。

## データ統合プロセス

### 1. データ収集

各データソースから個別にデータを収集しました。

### 2. 重複削除

同じ単語が複数のデータソースに存在する場合、以下の優先順位で採用しました：

1. 近畿日本語アクセント辞典（最も詳細）
2. Wikibooks 京阪式アクセント
3. 甲種日本語アクセント辞典

### 3. データクリーニング

- 空白行の削除
- アクセント記法の統一
- 品詞情報の整理

### 4. 品質管理

- 明らかな誤りの修正
- アクセント記法の一貫性チェック
